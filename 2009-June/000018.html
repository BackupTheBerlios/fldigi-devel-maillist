> If CMDPTT is used, what is the expected usage of these output pins?
> Leigh/WA5ZNU
>> 3.12AI
>>     RigCAT mods
>>             Added the following defaults to the xml file format:
>>                 <DTRPLUS>true/false</DTRPLUS>
>>                 <RTSPLUS>true/false</RTSPLUS>
>>                 <DTRPTT>true/false</DTRPTT>
>>                 <RTSPTT>true/false</RTSPTT>
>>         where setting ...PLUS true sets the corresponding control pin
>>         to +Vcc.  This is usually referred to as ON.
>> ...
>> Examples with new xml specifiers.
DTRPUS / RTSPLUS => usually used for powering a CIV interface that 
steals power from either RTS or DTR.
Some rigs also require DTR to be high before they communicate.

DTRPTT or RTSPTT used for a h/w PTT control usually with an NPN 
transistor similar to a 2N2222.

These controls would be used either in lieu of or in the case where a 
CAT PTT is not available.
