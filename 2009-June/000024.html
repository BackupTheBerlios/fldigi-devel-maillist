Here is the manual info; also see attached screenshot from PDF manual.

DTR and RTS: These are not used as serial I/O
handshaking lines. Instead, the K3 can use these as
PTT IN or KEY IN (see CONFIG:PTT-KEY). The
default for both signals is inactive. Refer to
application software documentation to determine if
it can use RS232 signal lines for PTT or keying.
If a PC or other device asserts RTS or DTR
while you’re in the PTT-KEY menu entry, the K3
will enter TEST mode as a precaution.
