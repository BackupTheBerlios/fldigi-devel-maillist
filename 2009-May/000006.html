In 3.12, a modal dialog window is displayed only the first time fldigi
logs a warning or error message.  Here is the comment block from the
part of the code that does this:

// If the backends logged an error and our stderr is not a tty,
// alert the user and offer to show the log window.  We only do
// this once and thereafter assume that the user will keep the
// log window displayed to be notified of errors.
