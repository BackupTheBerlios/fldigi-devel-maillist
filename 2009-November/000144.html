The QRZ CDROM data format is being retired and replaced by a new format
called QRZ 3.0.  We are actively soliciting comments on this new format.

The QRZ CDROM data format, first released in 1993, has a number of 
shortcomings
that have been addressed in the new format.

Software developers that support the QRZ CDROM format should have a look 
at this
new specification, sample source code, and file formats.

The specification is now in DRAFT status and we're requesting comments 
from the
developer community at this time.  There is also a developers package 
available with
sample data and C source code that demonstrates how the data can be 
searched.

Finally, a new Developer's Forum has been setup at QRZ that is open by 
invitation
only for developers to discuss topics related to the new format.  Just 
drop me an
email and I'll enable your membership.

Here are the links to the info:

QRZ 3.0 Specification
http://www.qrz.com/Developers/QRZdb3spec.pdf

QRZ 3.0 Sample Data and Source Code  (7.2 MB)
http://www.qrz.com/Developers/QRZ3Eval.zip

QRZ Developers Forum
http://forums.qrz.com/forumdisplay.php?f=59
