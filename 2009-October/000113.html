    RigCAT mods
    
        * on WIN32 platform - reinitialize commport when driver fails
          to be responsive.  Associated with some cheap USB dongle
          drivers on XP / Vista.
        * Modified serial i/o to accommodate transceivers that send
          unsolicited data streams - typical of Icom 7... series when
          transceiver CI-V port is set to "Transceive ON".
        * Corrected computed read after write delay.

Tested with my Icom-746PRO.  Code changes give excellent performance with

Retries - 2
Retry interval - 200
Write delay - 0
Baud rate - 19200
Stopbits - 1
Commands echoed - T
CAT command for PTT - T

Same performance whether Transceive is ON or OFF at the transceiver.  
Pre mod code would cause fldigi to loose serial comms to the transceiver.

73, Dave
