There is a new menu item under View to control the visibility of the 
upper (Shifted) set of macro buttons.

If visible you can toggle the upper set with Alt-2, Alt-3 and Alt-4 ... 
Alt-1 will give you Alt-2 ...
If hidden you can toggle the lower set with Alt-1 through Alt-4

The state of the embedded browser and the visibility of the upper macro 
button set, the log panel view etc are all saved between sessions.
