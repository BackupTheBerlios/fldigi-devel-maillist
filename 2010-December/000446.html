Remember that this is alpha code.  You can expect that things might not 
work as you expect and that you will fill in the missing documentation 
by experimenting.  All notices of releases for w1hkj software that 
contains a letter suffix is alpha code unless specifically noted otherwise.

Version 3.21.0AT - Alpha release

RTTY BROWSER
  Added code to reduce incidence of adjacent channel duplication
  Added reverse WF->browser link for rtty browser

Browser reformat
  Corrected line length computation after freq/ch# format change

PSK browser mod
  Modifications to detector
  Fix to mode change seg fault

ADIF format fault - back ported from 3.20.34
  Corrected formatting fault of ADIF output.  Fault caused when field 
data string exceeded 100 characters.  Fault prevalent on Windows 
platform by virtue of temporary ADIF file used to populate external log 
files.

In addition to the binaries and source tarball you will find a file 
named rtty-mixed.wav that you can use to test the RTTY browser.  It is 5 
RTTY Baudot-45 signals plus noise in a 3000 Hz passband.  There is also 
a file contributed by Skip, KH6TY, which I think is 20 meters in the 
late afternoon yesterday.  This is useful for testing the psk browser.

73, Dave, W1HKJ
