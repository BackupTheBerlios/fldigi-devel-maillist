You should notice considerably less noise in the psk browser.  I've been 
testing on 3580 with ARRL blasting away with CW in the meager 3 KHz 
spectrum used by PSK31 operators.  It does not trigger false 
detections.  CW is after all digital and it was there first.  Makes one 
wonder why the ARRL recommended PSK31 working spectrum falls right on 
top of the ARRL broadcasts.

Version 3.21.0AU
  Alpha release

PSK Browser-WF data
  * Changed WF FFT circular buffer to improve browser detector
  * Changed PSK channel timeout processing to reduce random channel noise.
  * Added false signal detection

GUI.  Hide the following tabs:
