Well here's an interesting bug (maybe)...

Using Windows XP, if you have the Hide/View Channels selected on when you
start up the program it will flash rapidly between the signal browser and
the normal screen making it impossible to see what your are doing.

The fix is to turn the Hide/View channel off, close the program, restart it,
and then turn it back on and both work normally but if you leave the
channels turned on and restart the program it starts the flashing again. As
I am not a programmer I don't know what is what with it but it's repeatable
on my system.
