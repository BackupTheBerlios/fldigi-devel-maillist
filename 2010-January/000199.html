 I have experienced a number of crashes in both the stable and alpha
 versions when fldigi has been left running for a long time (tens of hours,
 or several days). Nothing appears in the status log. Most of these crashes
 have occurred in my absence but I have noticed a possibly related tendency
 for the cpu load due to fldigi to rise sharply up to the point of zero
 idle time, followed by a crash. Yesterday I started running fldigi from
 the command line and this is what appeared in the terminal window after
 the first crash. It happened in my absence so I don't know whether the
 increased processor load happened this time. Alpha version compiled from
 scratch on Ubuntu 9.10, upgraded on 24th Jan 2010:
