modified/corrected since the first release of the document. All of these
appear to be reflected in changes to fldigi, except for the change to
ASCII 125 (0x7D) and its secondary character; therefore receiving or
transmitting this character correctly is not possible (it is confused
with 0x7B 123 '{').

understood the format of the "decoding table", so you should probably
