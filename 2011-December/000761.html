Please test compile the tarballs located at

http://www.w1hkj.com/developers/

These have been extensively modified to allow building the binary 
against either Fltk-1.1.10 or Fltk-1.3.0

The configure script tests for the Fltk version and sets conditional 
compilation flags accordingly.  I've tested test on two Ubuntu systems, 
one with Fltk-1.1.10 and the other Fltk-1.3.0.  I have not yet tested on 
a MinGW or an OS X build.

I would appreciate as many test builds against both Fltk versions.  If 
you experience any build problems please let me know which application, 
the Fltk version, and the OS.  You can ignore reporting any WARNING 
messages from the compiler.

Thanks.

73, Dave, W1HKJ
