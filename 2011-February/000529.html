    MT63 manual tune mod
   
      * Modified mt63 base and supporting digital signal processing (dsp)
        code to remove the fixed signal position restriction.
      * User can select either fixed 500 Hz lower tone signal format or
        to move the signal bandpass throughout the audio spectrum using
        the normal waterfall point and click

Please test and if you feel capable I would appreciate a technical 
review of the changes I made to the mt63base.cxx/h,  mt63.cxx/h and 
dsp.cxx/h source files.  I have asked Pawel Jalocha if he might want to 
review these changes but have not heard from him yet.  The MT63 dsp code 
is probably the most sophisticated of all the modem code that is in fldigi.
