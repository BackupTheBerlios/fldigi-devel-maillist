Version 3.21.0BB
      Alpha release

Maximize window error
   * Replaced HTgroup tiled widget with group widget
     - browser is fixed at 50% of HTgroup width
     - browser can be hidden
     - restart recognizes prior state of browser

I've given up (for the time being) on trying to get the tiled panel 
working on both Linux and Winders.  This version should work on all 
platforms independent of screen size and whether or not you have elected 
to have the main dialog be in full screen mode.  You can show and hide 
the embedded browser.  It is shown at 50% of the available horizontal space.

Let me know if you break it and how.
