NBEMS test suite has been upload to 

http://www.w1hkj.com/alpha/NBEMS/

This is a matched set.  It provides the capability to automatically view
incoming flmsg data streams on the system default internet browser.  I have
tested this on Ubuntu 10.04, Puppy, Vista and OS X (mini-mac G4).

You will find

*	source, dmg's, exe installation files, pets, and Linux binaries for
the two applictions.
*	images of captured data streams on Vista using Google Chrome as the
default browser (works ok with MS Explorer, but I do not like that app).
*	wav files that provide flmsg wrapped data streams for three
different modes.

Important changes

*	new NBEMS configuration tab under Misc
*	browser tab titles relate directly to either the contents of the
data stream, or
*	a reference to the stored file if the data stream was corrupted
