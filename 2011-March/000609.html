This is an NBEMS suite update.  These versions of fldigi and flmsg are 
keyed to each other to provide additional work flow functionality for 
NBEMS users.  The on-line documentation for both fldigi and flmsg have 
been updated with these changes.

http://www.w1hkj.com/FldigiHelp-3.21/ConfigMisc.html - see new NBEMS 
configuration tab
http://www.w1hkj.com/flmsg-help/index.html - new form ICS205A

download available at http://www.w1hkj.com/download.html

The principal change to the NBEMS suite is to allow a fully automated 
reception, decoding and display of flmsg data files.  Incoming files can 
be displayed using the system default browser.  Each new file is shown 
in it's own browser tab.  Files which are not received correctly will 
have an error message tab shown in the browser.

73, Dave, W1HKJ
