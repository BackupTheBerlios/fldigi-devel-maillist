Hi Al-

I recommend that you try the updated fldigi 3.21.34 version available
from the Ubuntu Hams group PPA:

https://launchpad.net/~ubuntu-hams-updates/+archive/ppa

To install that:

$ sudo apt-add-repository ppa:ubuntu-hams-updates/ppa
$ sudo apt-get update
$ sudo apt-get install fldigi

If that version still crashes, you could install the 'fldigi-dbg'
package (from the same PPA) and then run fldigi under the gdb debugger
to help us determine exactly where in fldigi its crashing.  Like this:

$ sudo apt-get install fldigi-dbg

$ gdb fldigi
(gdb) run
... wait for crash to occur ...
(gdb) where
... collect the output and post it here.

Feel free to contact me for further assistance with those procedures.
