Testing fldigi-3.21.35AA_setup.exe on WinXP SP3.

When installing on windows the Start Menu Shortcuts, Desktop Shortcuts and Quick Lauch Shortcuts are installed in current installer directory.
All shortcuts should be installed in "All Users" directory so all user accounts on that computer can see and use them.  Most window program installs do this.
This is for all windows installed programs like flrig, flmsg, flarq.

For example this is a problem for me because I use an Admin account to install/uninstall and a low level User account for running windows.
I manually move the shortcuts and delete them when a new release is out.
