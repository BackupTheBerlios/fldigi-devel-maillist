Hi,

There is a small mistake in the default name of the file containing all 
Navtex stations, for version 3.21.45.
It should be NAVTEX_Stations.csv instead of NAVTEX_Stations.txt.

The consequence is that the navtex decoder cannot display the emitting 
station.

To fix this, one just to go to the configuration tab for modems, choose 
Navtex and change the file. A permanent fix will come later, but once 
you have set your configuration, it is stable.

Thanks.
