I'm sorry Remi, but most of what you said just whizzed right over my head. - 
I am generally ignorant about Linux and struggle by with one of the 
simplified, compact versions ( Puppy ) on a formatted thumb-drive so I can 
carry it around and have an alternate intelligence for my little Acer XP 
netbook.

It works for me because it's so simple, a no brainer and it lets me use the 
Linux version of the NBEMS software. I always use it with the netbook - but 
I could carry the thumb drive around and boot up other computers with my 
NBEMS software as required.  Like if the netbook gets fried or the battery 
dies during a hurricane and I have to borrow my wife's notebook, for 
example.

During a hurricane, we typically have no internet or power. The phones go 
out here, not long after it starts a heavy rain. Installing and setting up a 
new computer with NBEMS software would be difficult if not impossible under 
those conditions ( no software downloads ) - and slow. Getting all of the 
parameters tweaked, etc.. But with the Puppy thumb drive, I can infect a new 
computer with already set-up NBEMS in just a few minutes. My callsign's 
already in there, my preferences set and my list of contacts and a few 
pre-composed messages are right there, ready to go.

That's not so bad!

My XP netbook has the NBEMS software's Windows version installed too and it 
runs just fine - but I use those programs for practice, and fun operating - 
while the Puppy Linux thumb-drive software is reserved for actual EMCOM use.

It helps me keep organized, and the Puppy Linux thumb-drive is not cluttered 
up with personal files, video games etc. which makes it much better suited 
to EMCOM work IMO.

The windows version of NBEMS under XP is used for casual EMCOM practice, for 
fun QSOs in Olivia 8/500 or PSK31, and has my operating log, etc. in it. - 
Stuff that I would hate to mix up with EMCOM.

- But that's about the extent of my knowledge about Linux. As I said, I'm 
generally ignorant about it, and don't know much about the nuts and bolts, 
etc... - The puppy Linux distribution really is best for guys like me.
