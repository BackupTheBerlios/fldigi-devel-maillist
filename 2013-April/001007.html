I pushed a version of fldigi (In my branch pu/rc) where the external 
library xmlrpc-c is replaced by the embedded livrary xmlrpcpp.

Consequences:
* Fldigi builds on Windows with MXE with the xml-rpc server.
* The code is simpler.
* Building on Mingw will be much simpler I guess.
