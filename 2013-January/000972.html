Version 3.21.66TTY

     Experimental TTY xmt/rcv implementation

       * This patch contributed to by
         W1HKJ, Dave Freese <w1hkj at w1hkj.com>
         DO2SMF, Stefan Fendt <smfendt at gmail.com>
       * Waveshaped FSK modem for classic RTTY using raised cosine 
matched filters
       * Kahn demodulator
       * RTTY scope based on separate mark / space filter outputs
       * Corrected s/n estimator
       * Change RTTY browser to new demodulator
       * Fixed missing browser highlight for callsign detection

     Status / Tx Level Controls

       * increased horizontal dimension to 120 pixels

Thanks to Ed, W3NR (on Linux) and Dick, AA5VU (on OS X) have been 
testing this version during it's development cycle.

The transmit signal is very clean and the Mark/Space waterfall signature 
is two narrow tracks.

RX decoder copies down to -9 dB.  The RX demodulator is continuing 
development, but this version is far superior to the current 3.21.66 
implementation.

The TTY browser also uses the new demodulator.  The range on the squelch 
has been changed to -9 dB to +9 dB.  Recommend setting it at -3 dB to 
start and then adjust according to band conditions.

Let us all know if you use 66TTY in this weekend's BARTG RTTY contest.

73, Dave, W1HKJ
