Just upgraded to Apple OSX Mavericks and wanted to share the process
required to get software compiling again from the OSX commandline.
The 10.9 update process will "break" the currently installed C / C++
compiler provided by Xcode, requiring an Xcode upgrade.

After installing the Mavericks 10.9 Update:

   1. Open the App Store
   2. Search for Xcode
   3. Update Xcode by clicking the update button
   4. Wait for Xcode update to finish installing
   5. Run Xcode & enter the administrator password when asked (to finalize
   commandline tools installation)
   6. After installation is finalized, exit Xcode


The C and C++ compiler now work from the OSX commandline again :-)
