Sorry, I hit the send button too hastily. This was in fact a CAT problem. I
was using an FT817 with a short write delay and this occasionally caused
the sideband to be incorrectly reported as LSB instead of USB. Fldigi then
compensated for this by inverting the spectrum.

I now have the following RigCAT settings:

Write delay: 100ms
Retry interval: 500ms
Baud Rate: 9600
Stop bits: 2

And all is now OK.
